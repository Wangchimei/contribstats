# ContribStats

A profile viewing app for checking contribution on Github.

## Build with

- React.js (Routers, Hooks, Context API)
- GitHub REST API

## Getting Started

- Install dependencies by running

  ```
  yarn
  ```

  or

  ```
  npm install
  ```

- Start a local server by running
  ```
  yarn start
  ```
  or
  ```
  npm start
  ```
